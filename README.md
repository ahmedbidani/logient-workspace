
## Installation

The app requires [Node.js](https://nodejs.org/) v18+ to run.

Install the dependencies and devDependencies and start the server.

```sh
npm i -g pnpm
cd logient-workspace
pnpm i
pnpm run dev
```
After that open the address of the app http://localhost:5173/
